# NourishMind AI

NourishMind AI is an accessible education platform designed to cater to diverse learning needs. 

## Project Setup

To run this project locally, ensure you have Node.js and npm/bun installed.

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/rameshtavishwakarma97-design/nourishmind-ai.git
   ```
2. Navigate into the directory:
   ```bash
   cd nourishmind
   ```
3. Install dependencies:
   ```bash
   npm install
   # or
   bun install
   ```

### Development Server

Start the application in development mode:

```bash
npm run dev
# or
bun run dev
```

## Technologies Used

- React (Vite)
- TypeScript
- Tailwind CSS
- shadcn/ui
- Supabase
